# Создание задачи

**Текст:** {{ user_text }}
**Очереди:** {{ ', '.join(available_queues) }}
**Приоритеты:** {{ ', '.join(available_priorities) }}

**Задача:** Создай структурированную задачу из текста пользователя.

**Формат ответа:**
```json
{
    "summary": "Название задачи (до 100 символов)",
    "description": "Подробное описание с контекстом",
    "queue": "DEV",
    "priority": "Высокий",
    "assignee": null,
    "deadline": "2025-07-06",
    "tags": ["bug", "auth"],
    "type": "bug"
}
```

**Правила:**
- **summary:** краткое название, что нужно сделать
- **description:** как, зачем, контекст, требования
- **queue:** семантический выбор из доступных
- **priority:** Критический > Высокий > Средний > Низкий
- **type:** bug, task, feature, epic
- **tags:** 2-3 релевантных тега

**Примеры:**
Вход: "Нужно исправить баг в авторизации"
Выход:
```json
{
    "summary": "Исправить баг в авторизации",
    "description": "На главной странице при попытке авторизации возникает ошибка. Проверить валидацию данных и логику аутентификации.",
    "queue": "DEV",
    "priority": "Высокий",
    "assignee": null,
    "deadline": null,
    "tags": ["bug", "auth"],
    "type": "bug"
}
```
